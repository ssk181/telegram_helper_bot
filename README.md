# **Telegram Helper Bot** #

## Установка

	apt-get install python3 python3-pip
	pip3 install -r requirements.txt 

## Регистрация бота в Telegram

Процесс регистрации бота в телеграмм через @BotFather описан здесь:
https://core.telegram.org/bots#botfather

После создания бота не забыть назначить ему права на просмотр сообщений в чате:

       /setprivacy

Полученный токен бота, вида **"110201543:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw"** нужно прописать в скрипте bot.py.

Также не забыть поправить CHAT_ID.
